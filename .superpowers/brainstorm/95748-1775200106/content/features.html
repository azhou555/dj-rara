Which features should DJ Rara have?

Select all that excite you — we'll figure out scope after

🌙

Mood Mode core

Pick a vibe (energetic, chill, focus, melancholy) and tune recommendations to match via Spotify audio features like energy, valence, tempo.

🎵

Browse & Curate core

Scroll through recommended tracks in the TUI, thumbs up/down individual songs before creating the playlist.

⏳

Time Period Selector

Switch between short-term (last 4 weeks), medium-term (6 months), and long-term (all time) taste profiles in the UI.

🎸

Genre Lens

Filter or bias recommendations toward specific genres. See a breakdown of your taste as a genre map.

▶️

Track Preview

Open a 30-second Spotify preview in the browser or system player directly from the TUI. Hear before you commit.

📜

Session History

Remember past DJ Rara playlists locally. Avoid re-recommending songs you've already received.

📊

Taste Stats Screen

A "Your Music DNA" view — top genres, audio feature averages (danceability, energy, etc.), most played artists visualized as ASCII bars.

📋

Playlist Manager

View, rename, or open existing DJ Rara playlists from the TUI. Don't just create — manage.

♪ The two core features will be included regardless. Select the extras you want.
