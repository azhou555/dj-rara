How should DJ Rara feel to use?

The interaction model shapes the whole experience

A

Guided Wizard low friction

Run python main.py and the CLI walks you through a series of gentle prompts — mood, genre vibes, how many tracks. No flags to memorize.

~ dj rara ~
────────────────
how are you feeling today?
[1] energetic [2] chill [3] focus [4] melancholy
> 2

how many tracks? (default 30)
> _
B

Live TUI Dashboard most fun

A full-screen interactive terminal UI (using rich). Browse recommendations live, toggle filters, preview tracks, confirm before creating playlist.

╭─ dj rara ─────────────────────────────────╮
│ ♫ Discovering for azhou ┊ 42 tracks │
├───────────────────────────────────────────┤
│ ▶ Neon Moon — Brooks & Dunn ♥ 84 │
│ Take It Easy — Eagles ♥ 91 │
│ Skinny Love — Bon Iver ♥ 79 │
╰───────────────────────────────────────────╯
[↑↓] navigate [space] toggle [p] playlist [q] quit
C

Flags + Beautiful Output power user

Familiar CLI flags like the current tool, but output is completely redesigned — rich progress bars, formatted track cards, animated spinners, color-coded results.
